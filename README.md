This project is based around creating a simple HTML website with internal links to a few recipes.
By the end of the project, I will have demonstrated the ability to create a simple website
with the boilerplate html elements and links.

Self reflection- During creation of the third recipe I realized I had forgotten to add alt attributes to the first two recipes.  I had to go back and fix this.  In addition, I noticed a user would have to use the browser back buttons to return to the home page so I added links at the bottom of each recipe to return to the homepage without the browser buttons.
